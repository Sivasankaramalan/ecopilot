## What EcoPilot does

EcoPilot is an MCP server that sits between your AI agent and the user. When the agent needs input, it calls a tool — no extra premium requests.

| Without EcoPilot | With EcoPilot |
|---|---|
| Agent asks follow-up → 1 premium request | `ask_user` → 0 premium requests |
| Verbose prompt → extra tokens | `tighten_prompt` → up to 60% fewer tokens |
| Wrong model selected | `suggest_model` → use mini or standard |
